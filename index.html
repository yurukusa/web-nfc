NFC Scanner Idle

Start scanning to read NDEF messages from NFC tags.

Generated Code

const ndef = new NDEFReader();

async function startScan() {
  try {
    await ndef.scan();
    console.log("NFC scan started.");

    ndef.addEventListener("reading", ({ message, serialNumber }) => {
      console.log(`Serial: ${serialNumber}`);
      for (const record of message.records) {
        console.log(`  TNF: ${record.recordType}`);
        console.log(`  Data: ${new TextDecoder().decode(record.data)}`);
      }
    });

    ndef.addEventListener("readingerror", () => {
      console.error("Cannot read data from NFC tag.");
    });
  } catch (error) {
    console.error(`Scan failed: ${error.message}`);
  }
}

NFC Writer Idle

Write the message built in the Builder tab to an NFC tag. Bring a tag close after pressing Write.

Generated Code

const ndef = new NDEFReader();

async function writeTag() {
  try {
    await ndef.write({
      records: [
        { recordType: "text", data: "Hello NFC" }
      ]
    });
    console.log("Message written.");
  } catch (error) {
    console.error(`Write failed: ${error.message}`);
  }
}

Make Tag Read-Only

⚠ Irreversible Operation

Making a tag read-only is permanent. The tag can never be written to again. This is implemented via NDEFReader.makeReadOnly(). There is no undo.

  • The current content on the tag will be locked forever.
  • Use this only for production/deployment tags.
  • Always verify the tag content before locking.

Generated Code

const ndef = new NDEFReader();

async function makeReadOnly() {
  try {
    await ndef.makeReadOnly();
    console.log("Tag is now permanently read-only.");
  } catch (error) {
    console.error(`Failed: ${error.message}`);
  }
}

NDEF Binary Format Reference

Byte 0: [MB][ME][CF][SR][IL][TNF TNF TNF]
Byte 1: TYPE_LENGTH
Byte 2: PAYLOAD_LENGTH (1 byte if SR=1, else 4 bytes)
Byte 3+: ID_LENGTH (if IL=1)
Then: TYPE ID PAYLOAD

Error Scenario Simulator

Simulate common NFC error conditions to test your error handling code.

Error Handling Code

async function safeNfcOperation() {
  try {
    const ndef = new NDEFReader();
    await ndef.scan();
    // ... operation ...
  } catch (error) {
    if (error.name === "NotAllowedError") {
      // User denied NFC permission
      // or feature policy blocks Web NFC
    } else if (error.name === "NotSupportedError") {
      // Browser or device doesn't support NFC
    } else if (error.name === "NotReadableError") {
      // Tag removed too quickly or read failure
    } else if (error.name === "NetworkError") {
      // Write failed — tag may be corrupted
    } else if (error.name === "AbortError") {
      // Operation was aborted
    } else if (error.name === "InvalidStateError") {
      // Tag is already read-only
    }
    console.error(error.name, error.message);
  }
}
